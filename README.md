## 💬 პროექტის შესაძლებლობები

* **📲 Home** – გაგზავნე შეტყობინება მომენტალურად **Enter** ღილაკზე დაჭერით. გაგზავნილი ტექსტები ეგრევე გამოჩნდება ინპუტის ქვემოთ ისტორიაში.
* **📊 Dashboard** – მართე ბაზაში არსებული ყველა მონაცემი. გამოიყენე **Select All** ფუნქცია რამდენიმე ჩანაწერის ერთდროულად მოსანიშნად და წასაშლელად.
* **💾 Cloud Sync** – ყველა მონაცემი ავტომატურად სინქრონიზდება დისტანციურ ბაზასთან (Cloud Functions).
* **⚙️ Data API** – პირდაპირი წვდომა ბაზის API-ზე Sidebar-ის ქვედა მარჯვენა კუთხიდან.

## 📥 როგორ გავუშვათ პროექტი (Local Setup)

ვინაიდან პროექტი ჯერ არ არის ჰოსტინგზე, მის სანახავად მიჰყევით ამ ნაბიჯებს:

1.  **გადმოწერეთ** პროექტის ფაილები (Source Code).
2.  გახსენით **VS Code** და შედით პროექტის საქაღალდეში.
3.  გახსენით ტერმინალი და ჩაწერეთ:
    ```bash
    npm install
    ```
4.  ბიბლიოთეკების დაყენების შემდეგ გაუშვით:
    ```bash
    npm run dev
    ```
5.  ბრაუზერში გახსენით მისამართი: `http://localhost:5173`
